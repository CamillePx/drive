Project Idea

Transforming an old toy into an On&Offscreen project… This is an interesting briefing.

Though I am a girl, I used to love playing with remote control cars when I was a child. Now that I am 21, I’d rather playing with technology. This project was for me the perfect opportunity to create my own remote control car. How exciting... and challenging for a newbie like me !

But God knows I love challenges. So let’s go !

How it works

First pair your smartphone with the car via Bluetooth.

Launch the app and connect it to the car.You can know make it move forward and backward, make it stop, make the horn work and the lights on and off.

These commands have all a specific number related. This number is sent via Bluetooth to the Arduino that interprets it into an action.

Have fun !

There were a lot of "first times" in this project. I had never worked with an Arduino or other electronics platforms (neither hardware nor software). I wasn't familiar at all with the bluetooth technology. I didn’t even know how to build an app.

My first step was the easiest one: the toy. I directly found this old red car, easy to open. But more interesting, it already has a cog fixed on its wheels axis. This couldn’t have been more perfect.

After that I focused on the hardware part. As an MVP I only wanted my car to be able to move (forward and backward) and stop. I did a lot of research on the Internet to learn how to use the Arduino and I finally got the hardware parts linked together, and the related Arduino code. I was then able to start and stop the motor using the Serial Monitor. I only add to write 0, 1 or 2 and the magic happened ! Once I’d really understood how it was working, I decided to add a horn and some headlights. I wanted my car to be really badass, you know?

Then I had to build the Android app to communicate with my arduino project. After some research on the Internet, I found a good basic one and adapted it to fit my purpose. This didn’t need to be really complicated actually. It only had to be able to connect via Bluetooth to the Arduino, and then send a specific number according to the button being clicked. For example, when you click on “Stop” the app sends a "0", understood by the Arduino as a "Motor Off".

Last but not least I had to put the whole system in the car, make it fit and fix the motor to the wheels axis. The power provided by one 9V battery wasn’t enough to make the car move, so I had to add another one. And it was working. Mission accomplished !
